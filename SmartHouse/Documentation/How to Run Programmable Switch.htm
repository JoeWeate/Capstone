Programmable Switch for NI myRIO
Choose Movement Consulting, LLC

Summary

The Programmable Switch for NI myRIO project demonstrates how to use the NI myRIO to wirelessly control a switch. This project is focused on controlling power to an electrical device that uses a standard power plug; for example, a lamp. This provides for many creative applications including home automation of lights or other electronics.

 

This document provides information on how to use the project and how to configure the circuitry to interface with a power source.

Software Requirements

This project requires:

      Standard NI myRIO software installation

      OpenG Toolkit (Installed automatically with Programmable Switch project)

      System Configuration software

 

To install the System Configuration software to your NI myRIO, complete the following:

1.     Open NI MAX.

2.     Expand Remote Systems and locate your NI myRIO

3.     Expand the NI myRIO, right-click on Software and choose Add/Remove Software

4.     If prompted for a password, simply click OK.

5.     Select Custom software installation.

6.     When prompted if this is what you want to do, click Yes.

7.     Locate NI System Configuration and NI System Configuration Remote Support

8.     Click the X next to each of these and select Install this feature.

9.     Click Next  and complete dialogs to install the software.

Hardware Requirements

This project requires the following:

      NI myRIO 1900 or 1950

      External relay circuitry to control power

      A smartphone (For smartphone control)


To control the power to a device, you must interface the NI myRIO to a relay and protection circuitry. There are many tutorials online for using solid-state relays to control AC power. In this document we will use a PowerSwitch Tail II, which is a low cost pre-made solution. If you choose to design your own relay circuit, take safety precautions.

 

Wiring the myRIO and relay:

Wire the (+) side of your relay to the DIO7 pin on Connector C of the NI myRIO.

Wire the (-) side of your relay to the DGND pin on Connector C of the NI myRIO.

 

DANGER:  Take extreme caution when working with live electricity. This can be very dangerous. For this reason, it is recommended to use a pre-made device like the PowerSwitch Tail II that implements protection circuitry and encloses wiring. Turn of all power when wiring connections.

Additional Notes

      All switch programs use 24-hour timestamps. For example, if you would like to set a program to turn on at 3:30 PM, you must create the program for 15:30.

      Make sure that the time has been set correctly in NI MAX before setting alarms. To confirm your settings are correct, follow the instructions in the Configuring Time on the NI myRIO section.

Configuring Time on the NI myRIO

You must configure the time on the myRIO in order to schedule actions correctly. To do this, do the following:

1.     Open NI MAX.

2.     Expand Remote Systems from the list of items on the left and select your NI myRIO.

3.     Click the Time Settings tab on the bottom of the page.

4.     Choose the appropriate time and date for your location. Be sure to select the appropriate Time Zone and Daylight Savings Time option.

5.     Click Save at the top of the page.

Instruction Protocol

The Programmable Switch for NI myRIO project enables you to send commands to the NI myRIO via UDP communication in order to do the following actions:

 

      Turn switch on

      Turn switch off

      Set and program times and days to trigger the switch on or off

      Synchronize time with your host computer

      Remove any programmed actions

      Replace an entire switch program schedule

 

All commands can be sent by any device with UDP Write capabilities including your computer or smart phone.

ON Command

Turns switch on.

 

UDP Write String:      on

OFF Command

Turns switch off.

 

UDP Write String:      off

SET Command

Sets a new switch program.

 

UDP Write String form:                      set HH:MM:SS <DAY> <ACTION>

Simplified UDP Write Strong form:   set HH:MM <DAY> <ACTION>

 

Where <DAY> is a 3 character abbreviation of the desired day of the week:

 

      eee     Every day

      sun     Sunday

      mon     Monday

      tue     Tuesday

      wed     Wednesday

      thu     Thursday

      fri     Friday

      sat     Saturday

 

<ACTION> is either on or off.

 

Example: set 10:42:15 mon on

This creates a program to turn the switch on every Monday at 10:42:15 AM. Switch programs use 24-hour timestamps.

REM Command

Removes a switch program.

 

UDP Write String form:                      rem HH:MM:SS <DAY> <ACTION>

Simplified UDP Write Strong form:   rem HH:MM <DAY> <ACTION>

 

Example: rem 10:42:15 mon on

This removes an existing program that turns the switch on every Monday at 10:42:15 AM. If this program does not exist, nothing will happen. Switch programs use 24-hour timestamps.

 

TIME Command

Sets the time on the NI myRIO.

 

To send the TIME command, you must flatten a LabVIEW Timestamp to string, and concatenate it to a time string.

 

UDP Write String form:          time <flattened LabVIEW Timestamp>

 

SWITCH Command

Replaces entire switch program schedule.

 

To send the SWITCH command, you must flatten a Switch Schedule control to string and concatenate it to a switch string.

 

UDP Write String form:          switch <flattened Switch Schedule>

Tutorial: Using the LabVIEW Project

This tutorial demonstrates how to use your computer to control and program the switch. Your NI myRIO should be configured and connected to your WiFi network.

 

1.     Take note of your computers IP Address

2.     Open RT Main.vi, found under the myRIO in the LabVIEW project

3.     Enter your computers IP Address in the Host IP input and click the Run arrow.

4.     Take note of your NI myRIOs IP address from NI Max (Network Settings tab)

5.     Open Desktop Main.vi, found under My Computer in the LabVIEW project.

6.     Enter the IP Address for your myRIO in the myRIO IP Address input and click the Run arrow.

7.     Click the On button. You will see LED0 turn on. If you have circuitry connected to the NI myRIO as described in the Hardware Requirements, it should turn on as well.

8.     Click the Off button. You will see LED0 turn off. If you have circuitry connected to the NI myRIO as described in the Hardware Requirements, it should turn off as well.

9.     Click the Sync button at the top right of the window. This should synchronize the time on the NI myRIO to the time on your computer. The time in the myRIO Time indicator should match your system time.

10.  Click Modify Schedule. This activates the Switch Schedule control. In the first row, change Action to On, enter a time in the form HH:MM:SS, or HH:MM, and choose a Day. Eee means every day.
> Note:Use 24-hour time format.

11.  Click Send Updates.

12.  When the myRIO Time reaches the Time you chose on the day you chose, the switch will turn on.

13.  Click Remove next to the first entry of the Switch Schedule to delete the switch program.

14.  Click the Off button. The switch will turn off.

15.  Click Stop on to stop the host VI from running.

16.  Click Stop in RT Main to stop the myRIO from running.

Troubleshooting

If the scheduled switch programs do not trigger at the desired time, make sure you are entering the time in 24-hour format (i.e. 3:15 PM = 15:15).
If you continue to experience issues, double-check the time settings for your myRIO in MAX, particularly if the correct daylight savings settings are selected. See the Configuring Time on the NI myRIO section for more information.


Tutorial: Controlling the Programmable Switch with Your Smartphone

 

The Programmable Switch can be controlled with any UDP communication, including the use of a smartphone. It does not require the Host Main VI to run.

 

Note: If the myRIO has just been turned on, it is likely that the time will need to be syncd. In which case run RT Main.vi and Desktop Main.vi, and click the Sync button. Now that the myRIO time has been syncd properly, you can stop and close Desktop Main.vi.

 

To control the programmable switch with your smartphone, do the following:

 

1.     Run RT Main.vi

2.     Download a UDP app for your phone. There are many free options. A couple examples for iPhone are Send UDP and UDP Tool.

3.     Go to the settings section of the app and input the following:

a.     IP Address: myRIO IP address

b.     Port: 61557

4.     Save the settings

5.     In the string input field of the UDP app, type on.

6.     Click send. The switch will turn on.

7.     In the string input field of the UDP app, type off.

8.     Click send. The switch will turn off.

9.     You can use your smartphone to send any of the commands listed in the Instruction Protocol section of this document. This includes turning the switch on (ON command) or off (OFF command), setting a switch program (SET command) and removing a switch program (REM command).
